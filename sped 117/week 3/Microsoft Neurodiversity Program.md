# Microsoft Neurodiversity Hiring Program

Source: https://www.microsoft.com/en-us/diversity/inside-microsoft/cross-disability/neurodiversityhiring

## Program Overview

Established in 2015, Microsoft's Neurodiversity Program has significantly impacted the company by attracting a diverse range of talent. Employees work on cutting-edge technologies including Windows, Azure, and Office 365.

The program offers an alternative front door into Microsoft while maintaining the same hiring assessment standards. What sets it apart is the support model provided to ensure all candidates can showcase their skills.

## Key Features

- Neuroinclusive design framework
- Proactive approach to creating supportive systems
- 10 years of insights embedded company-wide
- Monthly inclusive hiring webinars
